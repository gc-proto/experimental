How topography works

The default fonts and styled size settings make Canada.ca pages consistent and easily readable across devices. Visitors can adjust these defaults to suit their reading needs.

Line length

Constrain text line length to 65 characters. This ensures that no lines extend beyond a comfortable reading length.

Page layouts may be wider than 65 characters. The line length restriction applies only to characters in a line of text.

Link appearance

Links in the content area are underlined. The exceptions are:
  • links in the banner and footer

For full details, refer to: Links

HTML tags and style considerations

  • Bold: use sparingly for emphasis.
  • Italics: use The Canadian Style's rules. Don't use italics for emphasis, design, or decorative purposes.
  • Strikethrough: do not use the strikethrough element

The code: Design tokens

Design tokens are used to manage and render typography. They are open source and managed at Github.

Accessibility requirements

Typography considerations:

  • Open source
  • Noto Sans supports more than 800 languages, including indigenous languages
  • Passes the test to distinguish between “I”,”l”, and ”1”.

W3C recomendations

  • 1.4.12 Text Spacing [AA]
  • 1.4.4 Resize text [AA]
  • 1.4.6 Contrast (Enhanced) [AAA]
    • Minimum for AA is 3:1 contrast
    • AAA is enhanced contrast 7:1 except heading which can be 4.5:1
  • 1.4.8 Presentation [AAA]
