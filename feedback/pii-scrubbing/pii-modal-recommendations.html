Page feedback tool with PII data scrub

This page provides an example of the Page feedback tool using the Data scrub component.

The Data scrub component provides the ability to delete the Personal Identifiable Information (PII) from a form field before submission.

Here is the list of the PII that the plugin will search and remove:

  • Any 8 or more digits separated or not by the following characters: ' ', '/', '-', '.'
  • Canadian serial passport number
  • Email
  • Canadian postal code
  • Certain parametres considered non secured in a url

A popup message will inform the user if any fields contain personal information and will show the output of the scrubbed data. The user can choose to go back to the form and delete the data or to proceed with scrubbing the personal information automatically.
