MSCA Feedback Executive Dashboard

Analysis Date: July 9, 2026 Feedback Dataset

Executive Summary

User feedback is dominated by three critical issues: authentication barriers, difficulty accessing CPP/OAS information, and system reliability failures. Together, these issues account for roughly 77% of analyzed comments and directly prevent users from accessing services, viewing benefits, or completing transactions.

Priority 1: Authentication Experience

70 Comments (32%)

Observed Issues

  • Multi-factor authentication failures
  • SMS verification issues
  • Phone number conflicts
  • Registration failures
  • Account access loops

Executive Impact: Users are unable to access any online services and are diverted to call centres and Service Canada offices.

Priority 2: CPP/OAS Information Access

58 Comments (26%)

Observed Issues

  • Missing CPP estimates
  • Missing payment information
  • Missing OAS status information
  • No confirmation of submitted applications

Representative Feedback

I can't see any confirmation that my CPP will start in September or how much my payments will be.

Executive Impact: Users cannot verify benefit entitlement, payment amounts, or application progress.

Priority 3: Reliability & Technical Performance

42 Comments (19%)

Observed Issues

  • File upload failures
  • Pages freezing or timing out
  • Technical difficulty messages
  • Long load times
  • Error pages

Representative Feedback

File upload failed due to a connection problem. Try again later.
Website keeps freezing.
The system was experiencing technical difficulties.
Unable (after several tries on multiple days) to upload requested documents

Executive Impact: Users are unable to complete applications, upload documents, or retrieve information.

Secondary Issues

Account Management Challenges (13%)

  • Address changes unavailable
  • Banking updates unavailable
  • Phone number changes unavailable
  • Unable to manage tax withholding settings

CRA Expectation Gap (10%)

  • Users expected Notices of Assessment
  • Users expected TFSA and RRSP information
  • Users expected CRA integration

Executive Recommended Focus

  1. Resolve authentication and MFA barriers immediately
  2. Restore reliable access to CPP/OAS status and benefit information
  3. Address platform reliability, loading, and upload failures
  4. Improve self-service account management capabilities
  5. Clarify differences between MSCA and CRA services

Analysis Confidence

High Confidence
Feedback themes were highly repetitive and consistently clustered around authentication problems, benefit information access, technical failures, account-management tasks, and CRA-related expectations.
